## How to use
Follow these steps:
* Move into the folder ROS_todoList_chatbot/cogrob_ws
* chmod u+x src/rasa_ros/scripts/*
* catkin build
* source devel/setup.bash
* roslaunch rasa_ros dialogue.xml
